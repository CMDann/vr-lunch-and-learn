## Slide 26 — Safety First

**BEFORE YOU BEGIN**

### SAFETY FIRST

1. STAY WITHIN THE GUARDIAN BOUNDARY — IT'S THERE FOR A REASON
2. ONE SPOTTER PER VR USER — NEVER LEAVE THEM ALONE
3. TIE HAIR BACK. USE GLASSES MODE IF WEARING GLASSES.
4. SIT DOWN IF YOU FEEL DIZZY — DON'T PUSH THROUGH IT
5. TAP THE MAKER'S SHOULDER TO COMMUNICATE — NO SHOUTING
6. THE VR SPACE FEELS REAL — YOUR BODY IS STILL IN THE ROOM

## Slide 27 — Getting Started

**DEVICE ORIENTATION**

### GETTING STARTED

#### FITTING THE HEADSET
1. LOOSEN ALL STRAPS BEFORE PUTTING IT ON
2. SLIDE ONTO HEAD — SCREEN FACING FORWARD
3. TIGHTEN BACK STRAP FIRST, THEN SIDE STRAPS
4. ADJUST LENS SLIDER UNTIL IMAGE IS SHARP

#### CONTROLLER BUTTONS

| BUTTON | ACTION |
|---|---|
| TRIGGER | HOLD + MOVE TO PAINT |
| THUMBSTICK ← | SMALLER BRUSH |
| THUMBSTICK → | LARGER BRUSH |
| B BUTTON | UNDO LAST STROKE |
| BOTH GRIPS | GRAB + SCALE YOUR SCENE |
| MENU BUTTON | OPEN SETTINGS PANEL |

#### ORIENTATION + RESET

| INPUT | ACTION |
|---|---|
| HOLD MENU (2s) | RECENTER YOUR VIEW |
| TAP CONTROLLERS | SWAP PAINTING / PALETTE HAND |

**THE TOOL**

### OPENBRUSH

A FREE, OPEN-SOURCE VR PAINTING APP ORIGINALLY BUILT BY GOOGLE. BRUSHSTROKES FLOAT IN 3D SPACE — WALK AROUND AND BUILD FROM EVERY ANGLE. OPEN BASIC MODE FIRST FOR A SIMPLER INTERFACE.

#### BRUSHES — 60+ AVAILABLE

#### GETTING ORIENTED

**YOUR PALETTE** — Floats in your non-dominant hand. Point + trigger to select brushes.

**QUICK TOOLS** — Secondary button (painting hand). Eraser, dropper, re-color + more.

## Slide 29 — Activity 01: Draw Your Name

**OPENBRUSH ACTIVITY — 01 / 03**

### DRAW YOUR NAME

**GOAL:** Float your name in 3D space using OpenBrush.

| | |
|---|---|
| **TIME** | 10 — 15 minutes |
| **BRUSHES** | Ink, Marker, Light |
| **DIFFICULTY** | Beginner |

**Steps:**
1. Select the Ink or Marker brush from the palette
2. Use the trigger to draw each letter in front of you
3. Step back to view your name from a distance
4. Experiment with scale — try drawing huge or tiny
5. Switch to Light brush for a glowing effect

---

## Slide 30 — Activity 02: Create a Simple Object

**OPENBRUSH ACTIVITY — 02 / 03**

### CREATE A SIMPLE OBJECT

**GOAL:** Build a recognizable 3D form: house, tree, or animal.

| | |
|---|---|
| **TIME** | 10 — 15 minutes |
| **BRUSHES** | Geometry, Icing, Hypercolor |
| **DIFFICULTY** | Intermediate |

**Steps:**
1. Switch to Geometry or Icing for solid shapes
2. Think in 3D — you can draw on all sides of your object
3. Walk around your object as you build it
4. Add colour and texture with Hypercolor brush
5. Use the camera tool in OpenBrush to screenshot your scene

## Slide 31 — Activity 03: Collaborative Build

**OPENBRUSH ACTIVITY — 03 / 03**

### COLLABORATIVE BUILD

**GOAL:** Create one shared scene with the whole group.

| | |
|---|---|
| **TIME** | 5 — 10 minutes |
| **FORMAT** | Rotate headset |
| **DIFFICULTY** | Optional |

**Steps:**
1. Agree on a theme before anyone draws
2. Each person adds to the same scene — roles rotate
3. Suggested themes: TOWN, ALIEN LANDSCAPE, DREAM CLASSROOM
4. Navigator guides each maker toward empty space
5. Use the OpenBrush camera tool to save the final scene

## Slide 32 — Group Roles

**TEAMWORK**

### GROUP ROLES
ONE HEADSET AT A TIME — ROTATE AFTER EACH ACTIVITY

**MAKER** | IN THE HEADSET
Creating in OpenBrush. Stay within the Guardian boundary. Listen for Navigator guidance. Tap your own shoulder means stop.

**SPOTTER** | STANDS DIRECTLY BEHIND
Watches for real-world hazards at all times. Never let the Maker walk into anything. Tap their shoulder firmly to stop them.

**NAVIGATOR** | STANDS TO THE SIDE
Verbal direction only — "Turn left a bit", "Your palette is on your right". Also tracks time and calls the rotation.

**ON DECK** | WAITING + PREPARING
Watching from outside. Deciding what you want to create. Ready to rotate in when the Navigator calls time.
